## 核心结论

**小仓位试探，不宜重仓。**

| 项目 | 建议 |
|------|------|
| **林奇分类** | 困境反转型（不是稳定增长型） |
| **仓位** | 严格小仓位（总组合的3-5%以内） |
| **买入策略** | 不要一次性买入，分3批以上建仓。等待信号：季度亏损收窄/竞争补贴降温 |
| **关键监测指标** | ① 季度外卖单量市占率变化 ② 核心业务OPM何时转正 ③ 京东外卖的月烧钱速度和单量增速 |

| **止损线** | 若连续2个季度市占率加速下滑（如外卖份额跌破55%），说明"致命伤"而非"暂时受伤"，应止损 |

> **一句话总结**：美团的底子仍好，但当前处于"战时状态"——**赚钱的能力没消失，只是利润被竞争临时吃掉了**。问题是"临时"会持续多久。

## 多学科交叉验证

```
标的：美团（3690.HK）  类型：平台型（本地生活服务）

1. 经济学 | 母问题：经济结构是否支撑长期盈利？

         长期看佣金率可能从6-8%降至4-6%，压缩利润但不至于消灭。
   结论：⚠️

2. 心理学 | 母问题：市场情绪是否导致了错误定价？
   子问题：市场是否过度悲观？
   分析：2025年亏243亿是事实，但大部分是补贴竞争导致的"战时开支"
         而非商业模式崩塌。市场可能把"暂时亏损"当成"永久恶化"来定价。
   结论：✅（情绪可能过度悲观）

3. 生态学 | 母问题：竞争生态位是否稳固？
   子问题：美团是"顶级捕食者"还是正在被取代？
   分析：京东外卖日单500万 vs 美团日均2亿单（约40倍差距）。
         抖音在到店领域有冲击，但履约配送能力远不如美团。
         美团仍是绝对领导者，但从"独大"变成"一超多强"。
   结论：⚠️（仍是老大，但竞争格局恶化）

4. 概率论 | 母问题：期望值是否为正？
   子问题：不同情景的概率和赔率如何？
   分析：
   - 乐观情景（40%概率）：竞争2026年缓和→2027年利润恢复300亿→股价150-200。赔率：+80%~+140%
   - 中性情景（35%概率）：竞争持续→利润恢复到150-200亿→股价100-130。赔率：+20%~+60%
   - 悲观情景（25%概率）：京东+抖音持续烧钱→美团份额持续下降→股价50-70。赔率：-15%~-40%
   - 期望值 = 40%×110% + 35%×40% + 25%×(-27%) ≈ +51%
   结论：✅（期望值为正，但波动巨大）

5. 逆向思维 | 母问题：怎样会让我亏钱？
   子问题：美团怎样会失败？
   分析：
   ① 京东长期维持零佣金+社保补贴→不太可能，京东自己也亏不起
   ② 抖音建成自有配送网络→需要巨大投入和时间，短期难实现
   ③ 骑手社保成本成为长期负担→这是真实风险，年增20-80亿成本
   ④ 宏观消费持续疲软→外卖是刚需，受影响相对小
   最大风险：竞争长期不回归理性 + 社保成本刚性上升 = 利润中枢永久下移
   结论：⚠️（致命场景概率不高，但利润中枢下移是实际风险）

综合：2个✅  3个⚠️  0个❌
```
