## 一、个股估值工具选择（按林奇分类）

> 核心原则：**不同类型的公司，用不同的估值工具组合**。用错指标比不看指标更危险。

### 1.1 估值工具 × 林奇分类 对照表 ✅

| 林奇分类 | 主估值指标 | 辅助指标 | **绝对不要用** | 判断标准 |
|---------|-----------|---------|-------------|---------|
| **缓慢增长** | 股息率 | PE-TTM、PB | — | 股息率>5%→考虑买入；股息率降至3%以下→考虑卖出 |
| **稳定增长** | PE历史分位 + PEG | FCF收益率、DCF | — | PE<30%历史分位 + PEG<1→有安全边际 |
| **快速增长** | PEG | PS（市销率）、PE前瞻 | 静态PE（增速快导致PE看起来永远"贵"） | PEG<1→买入区间；PEG>1.5→偏贵 |
| **周期型** | PB + 行业供需信号 | 产能利用率、库存周期 | **PE（会严重误导！）** | PE最低时往往是行业顶部，PE最高/亏损时往往是行业底部 |
| **困境反转** | EV/EBITDA + 现金消耗率 | PS（亏损时PE无意义）、恢复后远期PE | 当前PE（亏损为负） | 有足够现金撑过困难期 + 恢复后远期PE合理→有投资价值 |
| **隐蔽资产** | NAV（净资产价值重估） | PB（账面净资产 vs 市价净资产） | PE（资产价值与盈利无关） | 市值<重估NAV的70%→有安全边际 |

> **理论依据**：达摩达兰（Aswath Damodaran，NYU Stern估值权威）在《Investment Valuation》中系统论述了不同公司生命阶段应使用不同估值方法。林奇分类法与公司生命阶段高度对应。 ✅

### 1.2 个股估值决策流程图 ✅

```
第3步质量筛选通过
        ↓
用林奇分类法定位公司类型
        ↓
┌───────────────────────────────────────────┐
│  查上方对照表 → 选择对应的主估值指标+辅助指标  │
│  ⚠️ 检查"绝对不要用"列，避免误用               │
└───────────────────────────────────────────┘
        ↓
计算/查询估值数据（数据来源见第三章）
        ↓
与判断标准对比 → 得出安全边际结论
        ↓
继续行动指南第4步后续：压力测试 + 综合决策
```

## 二、指数/ETF估值工具选择

> 核心原则：指数没有单一公司的商业模式和财报，估值逻辑与个股不同。关键是**选对指标类型**+**看百分位而非绝对值**。

### 2.1 指数类型 × 估值指标 对照表 ✅

| 指数类型 | 典型指数 | 主估值指标 | 辅助指标 | 不适用 | 说明 |
|---------|---------|-----------|---------|-------|------|
| **大盘宽基** | 沪深300、上证50、恒生指数、标普500 | PE-TTM百分位 | PB百分位、席勒PE（CAPE） | — | PE-TTM是最通用的宽基估值指标；CAPE适合判断美股长期估值水位 |
| **中小盘宽基** | 中证500、中证1000 | PE-TTM百分位 | PB百分位 | — | 中小盘盈利波动较大，PB可作辅助交叉验证 |
| **成长型指数** | 创业板指、科技龙头、半导体、CS新能车、纳斯达克100 | PE中位数百分位 | PS（市销率）百分位 | PB（轻资产高增长公司PB天然极高，无参考意义） | 部分成分股可能亏损导致PE失真，用中位数剔除极值；PS对高增长但暂未盈利的科技公司特别重要 |
| **红利/高股息策略** | 中证红利、上证红利、深证红利 | PE-TTM百分位 | **股息率**（核心辅助！） | — | 红利指数的本质是"买股息"，股息率>4%→较有吸引力；股息率<2.5%→可能偏贵 |
| **基本面策略** | 基本面50、基本面120 | PE-TTM百分位 | PB百分位 | — | 基本面筛选后的大盘股，PE是主流指标 |
| **金融行业** | 中证银行、证券公司 | **PB百分位** | 股息率（银行）、券商周期信号 | PE（金融机构利润波动大，PE不稳定） | 银行/券商是典型重资产+强周期金融，PB是核心；PB<1为常态（银行），不代表便宜也不代表贵 |
| **消费行业** | 中证消费、消费龙头、中证白酒 | PE-TTM百分位 | PE中位数 | — | 消费行业盈利稳定，PE最适用 |
| **医药行业** | 中证医药、300医药、全指医药 | PE-TTM百分位 | PE中位数 | — | 医药盈利相对稳定，PE适用 |
| **周期行业** | 中证军工、基建工程、农业种植 | PE-TTM百分位 | PB百分位 | — | 周期行业PE波动大，建议PE+PB交叉看 |
| **互联网/平台** | 中证互联网50 | PE-TTM百分位 | PS百分位 | — | 部分成分股可能亏损，PS作辅助 |
| **主题/跨行业** | 中证养老、中证环保 | PE-TTM百分位 | — | — | 跨行业主题指数用PE通用指标即可 |
| **港股中资** | 恒生国企指数 | PE-TTM百分位（正数等权） | PB百分位 | — | 含亏损股时用正数等权PE剔除干扰 |

> **理论依据**：中证指数公司官方发布的指数估值方法论 + 理杏仁等主流估值平台的指标选择逻辑。 ✅

### 2.2 指数估值百分位判断标准 ✅

| PE/PB百分位 | 估值判断 | 对应策略（配合行动指南ETF流程） |
|------------|---------|---------------------------|
| <20% | 极度低估 | 加大定投（平时2倍）或一次性加仓 |
| 20%-40% | 低估 | 正常定投，可适当增加 |
| 40%-60% | 合理 | 维持正常定投 |
| 60%-80% | 偏高 | 减少定投金额（平时一半） |
| >80% | 极度高估 | 暂停定投，考虑减仓 |

> **关键提醒**：百分位要看**5-10年区间**，太短（如1年）没有参考意义。

### 2.3 特殊估值指标说明

**席勒PE（CAPE / Shiller PE）** ⚠️

| 项目 | 说明 |
|------|------|
| 定义 | 当前股价 ÷ 过去10年经通胀调整的平均盈利 |
| 优势 | 消除了短期盈利波动，反映长期估值水位 |
| 适用 | 美股大盘指数（标普500、纳斯达克100）的**长期**估值判断 |
| 不适用 | A股（历史太短、行业结构变化大）、单一行业指数 |
| 当前参考 | 标普500 CAPE长期均值约17倍，>30倍→历史性高估区间 |
| 数据来源 | multpl.com、Robert Shiller官网 |

> ⚠️ 待你用实际数据验证当前CAPE值是否与数据来源一致。

**股息率** ✅

| 项目 | 说明 |
|------|------|
| 定义 | 过去12个月每股分红 ÷ 当前股价 |
| 优势 | 直接衡量"买入后每年能拿回多少现金"，是红利策略的核心指标 |
| 适用 | 红利类指数（中证红利、上证红利、深证红利）、高股息个股 |
| 判断标准 | >4%→有吸引力；3-4%→合理；<2.5%→偏贵（对红利指数而言） |
| 数据来源 | 理杏仁、中证指数官网、雪球 |

**PS（市销率）** ✅

| 项目 | 说明 |
|------|------|
| 定义 | 市值 ÷ 营业收入（TTM） |
| 优势 | 亏损公司PE无意义时，PS是唯一可用的"价格/规模"比率 |
| 适用 | 高增长但未盈利的科技公司、互联网平台（早期）、SaaS公司 |
| 局限 | 不反映盈利能力——收入高但永远不赚钱的公司PS再低也可能是陷阱 |
| 判断标准 | 需与同行业对比，无统一标准。PS百分位<30%→相对便宜 |
| 数据来源 | 富途、雪球、理杏仁 |

**EV/EBITDA** ⚠️

| 项目 | 说明 |
|------|------|
| 定义 | 企业价值（市值+净债务） ÷ 息税折旧摊销前利润 |
| 优势 | 排除了资本结构和折旧政策差异，更适合跨公司比较 |
| 适用 | 困境反转型、重资产行业（航空、电信、公用事业）、并购估值 |
| 判断标准 | 需与同行业对比。通常<10倍→偏便宜；>15倍→偏贵（行业差异大） |
| 数据来源 | 富途、GuruFocus、Simply Wall St |

> ⚠️ 待实战验证具体阈值在不同行业的适用性。

## 三、估值指标速查表

> 所有指标的一站式速查：定义、公式、适用场景、陷阱、数据来源。

### 3.1 完整指标速查 ✅

| 指标 | 公式 | 一句话含义 | 适用场景 | 常见陷阱 | 数据来源 |
|------|------|-----------|---------|---------|---------|
| **PE-TTM** | 市值 ÷ 过去12个月净利润 | 花多少钱买1块钱利润 | 盈利稳定的公司/指数 | 周期股利润高点PE最低≠便宜；亏损股PE为负无意义；一次性收益导致PE骤降 | 理杏仁、富途、雪球、东方财富 |
| **PE前瞻** | 市值 ÷ 未来12个月预期净利润 | 基于预期盈利的估值 | 快速增长型（反映未来而非过去） | 分析师预期经常过于乐观，建议打8折使用 | 富途、Bloomberg、Simply Wall St |
| **PEG** | PE ÷ 预期净利润增长率(%) | 估值是否配得上增速 | 稳定增长型、快速增长型 | 增速不稳定时PEG波动极大；负增长时PEG无意义 | 富途、雪球（需手动计算） |
| **PB** | 市值 ÷ 净资产 | 花多少钱买1块钱净资产 | 银行、保险、地产、重资产制造、周期股 | 轻资产/科技公司PB天然很高≠贵；净资产可能被低估（隐蔽资产）或高估（商誉减值风险） | 理杏仁、富途、东方财富 |
| **PS** | 市值 ÷ 营业收入(TTM) | 花多少钱买1块钱收入 | 亏损或早期高增长公司、互联网/SaaS | 不反映盈利能力——收入高但永不盈利的公司PS再低也危险 | 富途、雪球、理杏仁 |
| **股息率** | 每股分红(TTM) ÷ 股价 | 买入后每年收回多少现金 | 红利策略、缓慢增长型、高股息ETF | 异常高股息可能是特别分红（不可持续）或股价暴跌导致；要看分红连续性 | 理杏仁、中证指数官网、雪球 |
| **FCF收益率** | 自由现金流 ÷ 市值 | 公司每年"免费"产出的现金占市值比 | 盈利稳定的成熟公司 | 资本开支周期会导致FCF波动；需看3-5年平均 | 富途、GuruFocus |
| **EV/EBITDA** | 企业价值 ÷ EBITDA | 排除资本结构差异后的估值 | 困境反转、重资产、并购分析 | EBITDA忽略了真实的资本开支需求；高折旧行业EBITDA会美化实际盈利 | 富途、GuruFocus、Simply Wall St |
| **DCF** | 未来现金流折现总和 | 公司理论上"值多少钱" | 所有类型（作为参考区间） | "垃圾进垃圾出"——假设变1%结果变20%+；只给区间不给精确值 | Simply Wall St、GuruFocus、Alpha Spread |
| **席勒PE（CAPE）** | 股价 ÷ 10年通胀调整平均盈利 | 长期估值水位 | 美股大盘（标普500） | 行业结构变化大时失真（科技占比提升推高均值）；不适用于短期择时 | multpl.com、Shiller官网 |

### 3.2 "该用哪个指标？"快速决策树 ✅

```
标的是个股还是指数/ETF？
    │
    ├─ 个股 → 查第一章（按林奇分类选工具组合）
    │
    └─ 指数/ETF
         │
         ├─ 成分股大部分盈利？
         │    ├─ 是 → PE-TTM百分位（主）
         │    │       ├─ 是红利类？ → 加看股息率
         │    │       ├─ 是金融类？ → 改用PB百分位
         │    │       └─ 是美股大盘？ → 加看席勒PE
         │    │
         │    └─ 否（含较多亏损股）→ PE中位数百分位（主）+ PS百分位（辅）
         │
         └─ 不确定 → 查第二章对照表
```

## 五、估值数据来源汇总

> ⚠️ **重要提醒**：本表中的数据源是基于AI训练数据中的社区口碑和使用频率选出的，**并非经过系统性对比评测**。可靠度分级仅为逻辑推断，建议你在实际使用中验证数据质量。

### 可靠度分级说明

| 等级 | 含义 | 数据来源特征 |
|------|------|------------|
| ★★★ 官方/学术级 | 数据由编制机构或学术权威直接发布 | 无二次加工，可作为最终基准 |
| ★★☆ 专业平台 | 数据来自交易所披露，平台做标准化展示 | 数据可靠但展示方式可能有差异，建议交叉验证 |
| ★★ 社区首选 | 投资社区中使用频率高，数据经二次加工 | 方便易用但计算方法可能与官方有差异 |
| ★☆ 参考级 | 有独特功能但数据精度/模型假设未充分验证 | 用于辅助参考，不作为单一决策依据 |
