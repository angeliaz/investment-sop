## 两层决策体系

```
第一层：配什么？配多少？（资产配置框架.md）
  → 确定股票/债券/黄金/商品/现金的目标比例
  → 非权益资产（黄金/债券/商品）的买卖时机
  → 定期再平衡
        │
        ↓
第二层：在股票仓位里买什么？（个人投资决策行动指南.md）
  → 个股：五步决策流程 + 财报体检 + 多学科验证
  → ETF：三步简化版
  → 输出：投资建议报告 → output/
```

> **记住**：先做第一层（确保总资产分散），再做第二层（在股票仓位内选标的）。顺序不能反。
