# 投资学习路径 —— "霍华德式学习路径"

> **定位：学习建议与自我迭代指南**
> 
> 霍华德告诉我们，投资体系不是背出来的，而是**用出来的**。本文档提供系统化的学习路径，帮助你从输入→建立框架→实践迭代。

---

## 第一阶段：输入（1-3个月）

### 精读书单（按优先级排序）

| 优先级 | 书名 | 作者 | 核心收获 |
|--------|------|------|---------|
| ⭐必读 | 《投资最重要的事》 | 霍华德·马克斯 | 整合框架——把格雷厄姆、巴菲特、芒格、加尔布雷思融为一套决策流程 |
| ⭐必读 | 《聪明的投资者》 | 格雷厄姆 | 安全边际——不亏钱的逻辑，"市场先生"比喻 |
| ⭐必读 | 《穷查理宝典》 | 芒格 | 多元思维模型——跨学科分析、逆向思维、认知偏见 |
| ⭐必读 | 《原则：应对变化中的世界秩序》 | 达利欧 | 大周期与资产配置——理解债务周期、五大力量、全天候策略 |
| 选读 | 《彼得·林奇的成功投资》 | 林奇 | 生活选股——六种公司分类法，从消费者角度发现投资机会 |
| 选读 | 《反脆弱》 | 塔勒布 | 不确定性下的生存策略——杠铃策略、黑天鹅思维 |
| 选读 | 《安全边际》 | 卡拉曼 | 耐心是个人投资者最大优势——现金是子弹，什么都不做的价值 |

### 补充阅读

## 第二阶段：建立自己的检查清单（Checklist）

> 这个检查清单与行动指南的五步决策流程**本质相同**，但以"自问自答"的形式呈现，适合每次决策前快速过一遍。

每次投资决策前，回答以下问题：

```
□ 1. 我的总资产配置是否合理分散？（达利欧）
□ 2. 市场现在在周期的哪个位置？（加尔布雷思）
□ 3. 我有没有被情绪影响？（芒格-心理学）
□ 4. 这个资产的商业模式和护城河是什么？（巴菲特）
□ 5. 财报数据是否支撑我的质量判断？（财报体检五张卡）
□ 6. 当前价格是否提供了安全边际？（格雷厄姆）
□ 7. 如果我错了，最大亏损是多少？我能承受吗？（格雷厄姆+芒格逆向思维）
□ 8. 这个投资的期望值是正的吗？（芒格-概率论）
□ 9. 如果世界变坏了，我的组合还能活下来吗？（达利欧-压力测试）
```

## 第三阶段：实践与迭代

### 操作建议

1. **用小仓位实践上述框架** —— 不要一开始就重仓，先用小钱跑通整个决策流程
2. **每次投资后记录决策逻辑** —— 用行动指南的五步汇总模板记录，保存到 `output/` 目录
3. **定期复盘** —— 每季度回顾：哪些判断对了？哪些错了？错在哪一步？
4. **不断修正自己的判断偏差** —— 每次犯错都是迭代体系的机会

### 迭代节奏

| 频率 | 做什么 |
|------|--------|
| 每次决策 | 跑一遍五步流程，填写模板 |
| 每季度 | 复盘决策记录，检查认知偏见 |
| 每半年 | 回顾迭代优化池，补充体系不足 |
| 每年 | 重读1-2本核心书籍，用新经验重新理解 |
