## 两层决策体系

```
第一层：配什么、配多少？（资产配置框架.md）
  → 四层动态配置：股权 / 对冲 / 债券 / 现金
  → 月度快照机制 + 季度再平衡
        ↓
第二层：股票仓位里买什么？（投资组合SOP.md）
  → 个股：五步决策流程（宏观定位→定性→定量→否决门→汇总）
  → ETF：三步简化流程（宏观→产品红线→估值）
  → 输出：投资建议报告 → output/
```

> **顺序不能反**：先确定总资产配置（第一层），再选具体标的（第二层）。

---

## 核心原则

- **永不永久亏损**：第4步否决门任一红线命中 → 直接放弃
- **不懂不做**：无法用3句话说清楚商业模式/护城河/最大风险 → 直接放弃
- **数据不能靠AI编**：所有影响决策的数字必须来自真实数据源，并标注来源与日期
- **先定性再定量**：巴菲特质量门通过后，再用林奇分类法选估值工具
