> ### 🔴 AI反思机制（历史错误教训，AI必须在每次输出前自检）
>
> 以下是本项目协作中已发生的真实错误，AI在每次生成报告、给出建议、推荐数据源时，**必须逐条对照自检，避免重犯**：
>
> | # | 错误类型 | 历史教训 | 自检问题（AI必须自问） | 违反后果 |
> |---|---------|---------|---------------------|---------|
> | E1 | **虚假精确** | 美团报告填充了精确的股价82港元、亏损243亿等数字，用户误以为是实时数据 | "我即将输出的数字，来源是什么？是训练数据还是用户提供的？" | 用户可能基于过时数据做投资决策 |
> | E2 | **沉默填充** | 查不到数据时没有主动告知用户，而是直接用训练数据填充，且不标注 | "有没有任何数据项是我查不到/不确定的？我是否已全部标注？" | 用户无法区分可靠数据与AI猜测 |
> | E3 | **伪权威推荐** | 推荐数据源（理杏仁、富途等）时说得像经过评测，实际只是基于社区口碑 | "我推荐这个工具/来源/方法的依据是什么？是亲测还是听说？" | 用户信任不靠谱的数据源 |
> | E4 | **未验证的逻辑断言** | 修正估值指标（如创业板指PB→PE中位数）时表现得像是确定结论，实际无回测支撑 | "这个判断有实证支持吗？还是仅仅逻辑上说得通？" | 看似合理但实际无效的分析框架 |
> | E5 | **过度自信** | AI默认输出完整、自信的回答，不主动暴露不确定性 | "我的确定程度真的有这么高吗？有没有该说'我不确定'的地方？" | 用户误判AI输出的可靠程度 |
>
> **AI自检执行规则**：
> 1. **生成报告前**：逐条过E1-E5，任何一条命中 → 必须在输出中明确标注
> 2. **推荐工具/方法/来源时**：必须标注依据强度（`[亲测验证]` / `[逻辑推断]` / `[社区口碑]` / `[不确定]`）
> 3. **给出知识性判断时**：如果确定程度 < 80%，必须主动说明 "这一点我不够确定，建议你交叉验证"
> 4. **发现新的错误模式时**：主动提出添加到此表中，持续积累教训

## 零、投资哲学层级声明

### 0.1 核心纪律（不可违背）

> 这五条纪律优先于一切分析技巧。任何时候只要与纪律冲突，宁可不做。

| 纪律 | 一句话 | 在本SOP中的落地规则（硬约束） |
|------|--------|----------------------------|
| **Never lose money** | 目标是避免**永久性资本损失**，短期浮亏可忍 | 第2.5步「永久性损失防线」任一红线命中→直接否决；第4步必须有安全边际；任何结论不得突破第0步股权层上限 |
| **不懂不做** | 解释不清/验证不了/写不出失败路径，就不要下注 | 第2.5步能力圈红线；Pre-mortem写不出≥3条可验证失败路径→直接否决（先学习、别交易） |
| **Do the right things, do things right** | 投资方向正确，执行正确 | 优先选对公司、商业模式及管理层，再根据估值和纪律理性执行；先定性（第3步）再定量（第3.5-4步）；建立stop doing list; |
| **execute with patience and discipline** | 不强行出手，只在满足条件时行动 | 不满足硬门槛→耐心等待；满足但置信度不足→小仓试探；季度快照+再平衡机械执行（见「资产配置框架.md」） |
| **永远只投入可用金融资产** | 只用“即使回撤50%也不影响生活”的钱 | 本SOP仅对你的“可用金融资产”有效：先扣除应急金与未来1-2年确定性支出，再谈股权层仓位与对冲 |

### 0.2 工具层级声明（冲突处理）

> 本体系以**巴菲特价值投资**为核心框架，其他大师工具在此框架下承担特定功能角色。当工具之间指向不同方向时，以下层级决定谁优先。

```
第0层（生存约束）：达利欧 → 执行层见「资产配置框架.md」v3.0
  → 四层配置是硬约束：股权（0-95%）/ 对冲（0-15%）/ 债券（0-10%）/ 现金（残余）
  → 当前股权层上限由宏观象限 × 市场温度动态决定，每季度更新快照
  → 任何标的的分析结论都不能突破当前股权层上限

第1层（核心框架）：巴菲特
  → 决定：什么是好公司、什么价格可以接受、什么时候卖
  → 所有个股决策的最终裁判

第2层（否决层）：芒格
  → 逆向思维维度出现❌ → 直接否决，不进入后续步骤
  → 多个⚠️ → 仓位上限砍半

第3层（仓位控制层）：加尔布雷思 + 达利欧
  → 只影响总股票仓位上限、现金储备比例、以及新增资金节奏
  → 不作为个股卖出触发，不影响个股的质量/价格判断标准

第4层（估值工具层）：格雷厄姆
  → 提供DCF/PE/PB估值语言，用于排除荒谬定价
  → 安全边际阈值随公司质量调整，非固定折扣

地域修正层：中国公司专项核查
  → 港股/美国上市中概在第3步后强制执行
  → 影响仓位上限和折现率，不推翻质量判断
```

**冲突处理规则**：层级高的框架优先；两个同层工具方向相反时，预期仓位自动降低50%，标注为"存疑持仓"加强跟踪。**加尔布雷思的周期判断只调股权层上限/新增资金节奏，不作为个股卖出触发，也不改变个股质量/估值标准。**

## 二、逐一拆解五大核心逻辑

### 1. 格雷厄姆：安全边际 —— "我能承受多大的错误？"

**一句话本质**：不是赚钱的逻辑，而是**不亏钱的逻辑**。

**拆解为可操作要素**：

| 要素 | 含义 | 实操指标 |
|------|------|----------|
| 内在价值 | 资产真正值多少钱 | DCF估值、PB、PE合理区间 |
| 买入价格 | 你实际付了多少钱 | 当前市价 |
| 安全边际 | 价值 - 价格 = 缓冲垫 | 折扣率（如打7折买入） |

**核心公式**：  
> **安全边际 = 内在价值 - 买入价格**  
> 安全边际越大 → 容错空间越大 → 亏损概率越低

**关键认知升级**：
- 不需要精确计算内在价值（这不可能），只需要确认**价格明显低于价值**
- 安全边际不是"买便宜"，而是"为自己的无知买保险"
- **应用场景**：市场恐慌、行业利空、短期业绩不及预期时，往往出现安全边际

**"市场先生"比喻——理解市场波动的正确姿势**：

格雷厄姆假设你和一个叫"市场先生"的人合伙开公司。他每天来给你报价，要么买你的股份，要么卖他的给你。**问题是他情绪极不稳定**——高兴时报价离谱地高，沮丧时报价离谱地低。

| 错误做法 | 正确做法 |
|---------|---------|
| 他报高价→你兴奋跟着买 | 他报高价→考虑卖给他 |
| 他报低价→你恐慌跟着卖 | 他报低价→考虑从他手里买 |
| 把报价当成资产真实价值 | 把报价当成可利用的机会 |

> **生活类比**：水果店老板心情好时苹果卖3块，心情差时卖1块。苹果没变，变的是他心情。你应该在他心情差时多买几斤。
>
> **实战案例**：2022年腾讯跌到200港元，公司业务没有毁灭性变化，是"市场先生"极度悲观。把下跌当"公司变差"是跟随市场先生，当"打折买好资产"是利用市场先生。

### 2. 巴菲特：质量投资 —— "这个资产能不能自己长大？"

**一句话本质**：好资产会用时间帮你赚钱，差资产会用时间消灭你的本金。

**拆解为可操作的"质量三问"**：

| 维度 | 核心问题 | 判断标准 |
|------|----------|----------|
| 商业模式 | 靠什么赚钱？能持续吗？ | 收入是否可预测、可复制、可扩展 |
| 护城河 | 别人能轻易模仿吗？ | 品牌、网络效应、规模优势、技术壁垒、转换成本 |
| 管理层 | 掌舵人是否值得信赖？ | 过往业绩、资本配置能力、股东回报意识 |

**护城河四大标准详解与实例**：

| 护城河类型 | 一句话 | 典型案例 |
|-----------|-------|---------|
| **品牌** | 消费者愿意为品牌本身付溢价 | 茅台（同为白酒卖7倍溢价）、可口可乐（盲测输百事但销量赢）、爱马仕（越涨越有人买） |
| **网络效应** | 用户越多→产品越有价值→更多用户 | 微信（所有朋友都在，换不走）、美团（商家-骑手-用户三方飞轮）、CUDA（数百万开发者锁定英伟达生态） |
| **规模优势** | 规模越大→成本越低→小玩家活不下去 | 沃尔玛/Costco（采购量最大拿最低价）、台积电（一条3nm产线200亿美元）、AWS（边际成本碾压小厂） |
| **转换成本** | 用户想换但换不动 | 苹果生态（全家桶迁移成本极高）、Adobe（设计师10年工作流绑定）、SAP（企业部署3年+数千万，换=重来） |

> **口诀**：品牌让人"愿意多付钱"，网络效应让人"离不开"，规模让对手"做不起"，转换成本让用户"换不动"。

**核心认知升级**：
- "便宜的垃圾"永远比不上"合理价格的优质资产"
- 质量投资的核心是**复利**：优质资产的内在价值本身在增长，时间是朋友
- **复利公式**：终值 = 本金 × (1 + 增长率)^时间。增长率哪怕只高几个百分点，长期差异是天文数字

### 3. 芒格：跨学科思维模型 —— "我是不是只看到了一面？"

**一句话本质**：用多把钥匙开同一把锁，比用一把钥匙死磕更有效。

**拆解为投资中最实用的5个跨学科模型**：

| 学科 | 核心模型 | 投资应用 |
|------|----------|----------|
| 心理学 | 锚定效应、从众行为、损失厌恶 | 理解市场为什么会过度反应（暴涨暴跌） |
| 概率论 | 期望值、大数定律 | 不追求每次都对，追求长期期望值为正 |
| 物理学 | 临界点、均值回归 | 极端行情终会回归，泡沫和恐慌都不持久 |
| 生物学 | 进化与适应、生态位 | 企业竞争如生态竞争，适者生存 |
| 哲学 | 逆向思维（反过来想） | "怎样会失败？" 比 "怎样会成功？" 更重要 |

**锚定效应详解——投资中最隐蔽的陷阱**：

人在判断时会被**第一个接触的数字"锚住"**，后续判断不自觉地围绕它微调，即使这个数字毫无逻辑关系。

| 锚定类型 | 表现 | 风险 |
|---------|------|------|
| 成本价锚定 | "我50块买的，等回50就卖" | 你的成本和股票价值毫无关系，股票不会因为你50买的就涨回50 |
| 历史高点锚定 | "美团最高400，现在150好便宜" | 400可能是泡沫价，150可能仍偏贵。应和内在价值比，不和历史高点比 |
| 整数锚定 | "茅台跌破1500！""突破2000！" | 1500和2000只是整数，和公司价值无关，但大脑天然对整数敏感 |

> **对抗方法**：问自己"如果今天第一次看到这只股票，完全不知道历史价格，我愿意以当前价买入吗？"答案为否，你可能只是被锚住了。

**多学科交叉验证——母问题与子问题框架**：

每个学科有一个**不变的母问题**（放入模板），但**具体子问题随标的类型变化**：

| 学科 | 母问题（不变） | 子问题随标的变化的示例 |
|------|-------------|-------------------|
| 经济学 | 经济结构是否支撑长期盈利？ | 平台型→佣金率能维持吗？硬件型→定价权来自技术还是垄断？消费品→提价能力多强？ |
| 心理学 | 市场情绪是否导致了错误定价？ | 暴跌资产→损失厌恶导致非理性抛售？热门概念→FOMO驱动追涨？长期低迷→坏消息被过度放大？ |
| 生态学 | 竞争生态位是否稳固？ | 平台型→是顶级捕食者还是中间物种？新兴行业→"寒武纪大爆发"阶段赢家未定？衰落行业→能进化出新能力吗？ |
| 概率论 | 各情景的期望值是否为正？ | 高估值资产→上行下行是否对称？困境反转→低概率高赔率该投多少？分散组合→持有10个正期望标的整体亏损概率？ |
| 逆向思维 | 怎样会让我亏钱？ | 平台型→被完全替代的概率？热门股→不需要变差只需增速不及预期？自己特别看好的→"我为什么可能是错的？" |

**多学科交叉验证操作模板**：

```
标的：______  类型：______（平台型/硬件型/消费品/周期股/指数/其他）

**核心认知升级**：
- **反过来想**：不问"这个投资能赚多少"，先问"这个投资怎样会让我亏光"
- **概率思维**：每个投资都是一次下注，关键是**赔率×概率**的期望值是否为正
- **消除偏见**：当你发现自己特别确信某件事时，恰恰要警惕

### 4. 加尔布雷思：金融狂热心理学 —— "现在大家都在想什么？"

**一句话本质**：市场不是经济的温度计，而是**人性的放大器**。

**拆解为"周期四阶段"判断模型**：

```
        乐观 ──────→ 狂热
        ↑                ↓
     复苏/修复        崩溃/恐慌
        ↑                ↓
        冷静 ←────── 绝望
```

| 阶段 | 市场特征 | 大众行为 | 正确策略 |
|------|----------|----------|----------|
| 冷静→乐观 | 估值合理，慢牛启动 | 半信半疑 | 逐步建仓 |
| 乐观→狂热 | 估值泡沫，人人谈股 | "这次不一样" | 逐步减仓 |
| 狂热→崩溃 | 暴跌、流动性枯竭 | 恐慌抛售 | 准备子弹 |
| 崩溃→冷静 | 遍地便宜货，无人问津 | "再也不碰" | 大胆买入 |

**核心认知升级**：
- **周期判断的核心信号不是数据，而是"人"**：出租车司机给你推荐股票→狂热；专业投资者说"再也不碰"→绝望
- 大众共识几乎永远是错的极端判断
- **霍华德的核心贡献**：他把加尔布雷思的定性描述变成了可操作的周期定位方法

### 5. 达利欧：全天候生存 —— "如果世界变坏了，我的组合还能活吗？"

**一句话本质**：不要试图预测未来，而是**构建一个无论未来如何都能存活的组合**。

**核心框架：五大力量 × 全天候配置**

达利欧认为，驱动世界变化的根本力量有五个：

| 力量 | 含义 | 对投资的影响 |
|------|------|-------------|
| 债务与货币周期 | 长期信贷扩张→债务过高→危机 | 决定利率方向、货币贬值风险 |
| 社会内部冲突 | 贫富分化→政治极化→政策剧变 | 影响税收、监管、资本流动 |
| 大国地缘博弈 | 中美竞争、国际秩序重构 | 影响供应链、汇率、市场准入 |
| 自然灾害与气候 | 极端天气、疫情等黑天鹅 | 短期冲击供应链和市场情绪 |
| 技术革命（AI） | 唯一持续向上的力量 | 重塑产业格局，创造新赢家 |

**拆解为可操作要素**：

| 要素 | 核心原则 | 实操方法 |
|------|---------|---------|
| 多元化分散 | 持有10-15个低相关资产 | 跨资产类别（股票/债券/黄金/商品） |
| 反法币配置 | 配置10-15%"对抗货币贬值"资产 | 黄金为首选，比特币为补充 |
| 风险平价 | 每类资产贡献等量风险，而非等量资金 | 波动大的少配、波动小的多配 |
| 再平衡纪律 | 定期恢复目标比例 | 每季度检视，偏离>5%时调整 |

**全天候策略参考配置**（达利欧原版，供理解分散思想用）：

| 资产类别 | 配置比例 | 逻辑 |
|---------|---------|------|
| 股票（宽基指数） | 30% | 经济增长期的主要收益来源 |
| 长期债券（20-25年） | 40% | 经济衰退期的保护垫 |
| 中期债券（5-10年） | 15% | 通胀上升期表现良好 |
| 黄金 | 7.5% | 乱世避险、对抗货币贬值 |
| 商品 | 7.5% | 通胀期的收益补充 |

> **⚠️ 本框架不照搬此配置。** 达利欧的重债券/轻股票比例与巴菲特股权长持哲学不兼容。本框架采用四层全动态结构（股权0-95%/对冲0-15%/债券0-10%/现金残余），由宏观象限×市场温度动态决定。→ **见「资产配置框架.md」v3.0**。
>
> 达利欧全天候的核心价值在于**思想**："分散+再平衡"；具体比例不直接套用。

**宏观观点输入（可选，但必须引用来源）**：

**关键认知升级**：
- **先确保生存，再追求收益**——与格雷厄姆"安全边际"思想在组合层面的呼应
- 不要把所有钱都拿去买股票，无论你多看好某个标的
- **定投+再平衡**是普通人对抗宏观不确定性的最佳武器
- "假设世界会变坏，你的组合还能活下来吗？"——这是每个投资者都必须回答的压力测试

## 三、霍华德的整合方法论 —— "五维融合"实操框架

霍华德的天才不在于发明，而在于**融合**。结合达利欧的宏观视角，完整的投资决策流程可以简化为：

```
第0步（达利欧 → 资产配置框架.md v3.0）：
  当前四层配比是否已确定？股权层上限是多少？对冲层有没有配？
  → 在股权层上限内才执行后续步骤
        ↓
第1步（加尔布雷思）：现在市场在周期的哪个位置？
  注：周期只调股权层上限/新增资金节奏，不作为个股卖出触发
        ↓
第2步（芒格）：多角度分析，排除自己的认知偏见
        ↓
第2.5步（永久性损失防线）：先排除“会死”的风险（硬门槛）
        ↓
第3步（巴菲特）：在这个位置上，哪些是真正优质的资产？（定性）
        ↓
第3.5步（财报体检）：数据是否支撑你的定性判断？（定量）
  → 取数完成后，回到第2.5步补做「取数后红线」（生存/财报可信度）
        ↓
第4步（格雷厄姆）：这些优质资产的价格是否提供了足够的安全边际？（估值）
        ↓
    全部满足 → 在组合允许的仓位内重仓买入
    部分满足 → 小仓位试探
    都不满足 → 耐心等待，但ETF底仓按你的配置框架；黄金按对冲层信号决定是否持有
```

### 五步决策流程实操指南（含“永久性损失防线”硬门槛）

**第1步：周期定位——"现在在哪？"**

| 观察维度 | 乐观/狂热信号 | 冷静/恐慌信号 |
|---------|-------------|-------------|
| 身边信号 | 非投资圈的人开始聊股票 | 专业投资者说"再也不碰" |
| 媒体信号 | 头条充斥"新时代""这次不一样" | 头条充斥"崩盘""史上最差" |
| 估值信号 | PE处于历史80%以上分位 | PE处于历史20%以下分位 |
| 资金信号 | 新基金火爆、打新热 | 基金清盘、赎回潮 |
| 恐慌指标 | VIX<15（极度乐观） | VIX>30（极度恐慌） |

> 输出一句话判断，如："当前港股处于冷静→乐观初期"。

> 注：周期定位只用于调整股权层上限/新增资金节奏，不作为个股卖出触发，也不改变第3-4步的质量/估值标准。

**第2步：认知校准——"我有没有偏见？"**

过三关：

| 关卡 | 自问 | 目的 |
|------|------|------|
| 锚定关 | 我的判断是否受了某个数字/事件的过度影响？ | 排除被历史价格、某条新闻锚住 |
| 从众关 | 我的观点和大多数人一样吗？如果一样要格外小心 | 大众共识在极端时几乎永远错 |
| 确认关 | 我是不是只在找支持我观点的证据？ | 逼自己找反面论据 |

> 实操：写下判断后花10分钟**只找反对理由**。找不到有力反对理由本身就是危险信号。

> **芒格否决规则（具有一票否决权，优先于一切后续步骤）**：
> - **逆向思维维度（"怎样会让我亏钱"）得出❌** → 直接否决，不进入第3步；记录否决原因，情况改变后可重新进入流程
> - **5个维度中出现3个及以上⚠️** → 不否决，但后续仓位上限自动砍半
> - **任一关卡未通过（锚定/从众/确认）** → 强制重做该关卡，而非直接继续

> 通过第2步不代表可以买入。进入第3步前，必须先执行第2.5步的「快速红线」；在第3.5步取数后，必须补做「取数后红线」（任一命中均否决）。

**第2.5步：永久性损失防线（硬门槛）——"会不会永久亏钱？"**

> 目标：避免**永久性资本损失**，而不是避免短期回撤。任何一条红线命中 → **直接否决**（不做估值、不谈仓位）；信息不足无法判断 → **暂停**（不买，先补齐信息）。
>
> 执行方式：先做「2.5.1 快速红线」（不依赖财报取数）；完成第3.5步取数后，再做「2.5.2 取数后红线」（生存/财报可信度）。两类任一命中 → 直接否决。

### 2.5.1 快速红线（监管/诚信/结构性，命中任一条 → 直接否决）

| 红线 | 典型信号（可观察） | 为什么致命 | 动作 |
|------|------------------|-----------|------|
| **能力圈红线（不懂不做）** | 不能用3句话说清：怎么赚钱/护城河/最大风险；关键变量无法验证 | 你无法在坏消息中保持理性，会用情绪交易 | 直接否决（先学习，别下注） |
| **信任红线（管理层/大股东）** | 重大欺诈/造假证据；长期利益输送/关联交易侵害小股东；资本配置明显反复“坑股东” | 信任断裂不可修复，护城河与财报都失去意义 | 直接否决（已持有→按卖出框架处理） |
| **监管/牌照红线（非线性事件）** | 进行中的重大监管调查/牌照可能被撤；政策明确逆风且商业模式依赖政策红利 | 不是“折现率+1%”能覆盖的风险 | 直接否决 |
| **结构性/地缘红线（非经营风险）** | 已进入退市/制裁/资本流动限制等程序；或关键风险无法做压力测试且你无法接受最坏结果 | 不可控，且常在极端行情放大 | 直接否决 |

### 2.5.2 取数后红线（生存/财报可信度，命中任一条 → 直接否决）

| 红线 | 典型信号（来自财报/公告/审计） | 为什么致命 | 动作 |
|------|----------------------------|-----------|------|
| **财报可信度红线（数字不可信）** | 审计保留/否定/无法表示意见；频繁更换审计师；利润与现金流长期背离且无合理解释 | DCF与指标全部失效，“便宜”可能是陷阱 | 直接否决 |
| **生存红线（融资断粮风险）** | 现金覆盖<12个月且现金流持续为负；短债压力大/利息覆盖过低；高度依赖再融资才能活 | 熊市里最容易直接归零 | 直接否决 |

### 2.5.3 Pre-mortem（永久性损失推演，必填）

> 如果你写不出失败路径，通常意味着你并不真正理解这家公司——这等同于“能力圈红线”。

```
永久性损失 Pre-mortem（必填）
标的：
核心论点（1句话）：

我可能永久亏钱的5种方式（至少写出3条“可被事实验证”的路径）：
1) 失败路径：
   先行信号（能观察/能查的数据或事件）：
   触发动作（冻结加仓/减仓/清仓/回到第3步重做质量筛选）：
   数据来源与监控频率：
2) ...
```

> **熊市加仓特别规则**：市场整体下跌≠公司更安全。熊市中任何加仓前，必须重做本Pre-mortem，并重点复查第3.5步的**现金/负债/竞争**三张体检卡，确认“活得下去”。

**第3步：质量筛选——"这是好资产吗？"**

过三关：

| 关卡 | 核心问题 | 通过标准 |
|------|---------|---------|
| 商业模式关 | 靠什么赚钱？10年后还能这样赚吗？ | 收入可预测、可复制、可扩展 |
| 护城河关 | 给竞争对手100亿，能复制这家公司吗？ | 至少有1条护城河（品牌/网络/规模/转换成本） |
| 管理层关 | CEO过去5年做了什么？钱怎么花的？ | 利润用于回购/分红/有效投资，而非乱花 |

**第3步补充工具：林奇六种公司分类法**

> 质量筛选通过后，用林奇分类法**快速定位公司类型**，不同类型对应不同的操作策略、估值方法和预期回报。

| 类型 | 特征 | 典型案例 | 买入策略 | 卖出策略 | 预期回报 |
|------|------|---------|---------|---------|---------|
| **缓慢增长** | 增速≈GDP（2-4%），高分红，规模大历史久 | 长江电力、大秦铁路、公用事业 | 股息率高于5%时考虑 | 股息率降低 | 年化4-6%（股息为主） |
| **稳定增长** | 大公司，增速10-12%，抗周期 | 腾讯、苹果、可口可乐、宝洁 | PE低于历史中位数时买入；DCF基准情景折扣15-25% | **护城河被永久破坏 / 管理层失去信任 / 初始论点被证伪 / 估值高于DCF乐观情景30%以上**；"涨了多少"永远不是卖出理由 | 年化10-15% |
| **快速增长** | 中小公司，增速20-25%，扩张期 | 拼多多（早期）、英伟达（AI周期） | PEG<1时买入，资产负债表健康 | **仅卫星仓位适用**：确认“增速下台阶”（见下方信号定义）→ 立刻清仓/大幅减仓；PEG>1.5偏贵 → 不加仓/考虑减仓 | 年化20-50%+（波动极大） |
| **周期型** | 盈利随行业供需周期大幅波动 | 万科、中远海控、中国神华 | 存货减少+价格回升=买入信号。**不看PE** | 存货增加+价格下降=卖出信号 | 单次波段30-100% |
| **困境反转** | 曾经的好公司遭遇重大打击，可能恢复 | 2022年Meta、瑞幸（退市后重组） | 确认"暂时受伤"而非"致命伤"，有现金能撑过去 | 股价恢复到困境前合理估值附近 | 翻倍+（成功），归零（失败） |
| **隐蔽资产** | 拥有被市场忽略的值钱资产 | 账上现金>市值的日本小公司、持大量土地的老国企 | 发现隐蔽资产被严重低估时买入 | 隐蔽资产不再隐蔽（被发现/分拆/收购） | 不确定 |

**快速增长型： “增速下台阶”信号定义（仅卫星仓位卖出触发）**

> 快速增长的核心论点通常就是“高增速可持续”，且估值对增速极其敏感；因此卖出纪律更严格，但**不允许用单季度噪音触发**。判定必须基于可查数据/公告/指引，并标注 `[来源,{YYYY-MM-DD}]`。

满足任一条，视为“增速下台阶”（触发清仓/大幅减仓）：
- `{核心增长指标}`（如核心营收/用户/交易额/订阅等）同比增速**连续2个季度**下滑，且公司指引/一致预期同步下修
- 增速下滑伴随**结构性恶化**：毛利率/经营利润率/自由现金流质量走弱，且管理层解释指向竞争/商业模式变化（而非一次性因素）
- **竞争格局逆转**：市场份额持续流失、核心产品被替代、价格战导致单位经济不可逆变差

不触发（常见误判，动作是“回到第3步+第3.5步重评”，而不是立刻卖）：
- 单季度不及预期/基数效应/行业短周期扰动，但护城河与长期需求未被证伪

**个人投资者仓位配置建议**：

| 仓位类型 | 适合的分类 | 说明 |
|---------|-----------|------|
| **主力仓位** | 稳定增长型 | 判断简单、容错高，长期持有为主，基本面破坏才卖 |
| **卫星仓位** | 快速增长型 | 用生活体验发现标的，PEG<1时介入，小仓位 |
| **机会仓位** | 困境反转型 | 仅限曾经的好公司暂时遇难，严格小仓位 |
| **ETF替代** | 缓慢增长型 | 高股息ETF比个股更省心，林奇自己也不买这类 |
| **不碰** | 周期型、隐蔽资产型 | 周期型需行业内经验，隐蔽资产型信息壁垒太高 |

**第3步补充：中国公司专项风险核查（港股 / 美国上市中概必做）**

> 巴菲特框架建立在可预测的长期竞争格局之上。以下风险是西方框架未覆盖的中国特有维度，需在林奇质量筛选通过后单独核查。结果影响仓位上限和折现率，不推翻质量判断本身。

> **红线优先**：若命中第2.5步的任一红线（如进行中的重大监管调查、管理层重大欺诈、审计无法表示意见、退市程序启动等）→ **直接否决**，不进入本表的“仓位/折现率修正”讨论。

| 维度 | 核查问题 | 风险信号判定 | 仓位/折现率影响 |
|------|---------|------------|--------------|
| **监管政策风险** | 所在行业过去5年是否经历重大整治？政策方向是否明显不确定？ | 平台经济/游戏/教育/数据安全等敏感领域；历史有强监管/多轮整治 | 仓位上限×60%；折现率+1.5% |
| **管理层信任度** | 创始人/核心管理层是否仍在位？过去3年是否有关联交易、高管大比例减持或关键人物突然离职？ | 创始人离开+无明确接班人；有大比例减持记录 | 仓位上限×80% |
| **股东回报记录** | 是否有实质性回购/分红执行记录（非仅宣布计划）？过去3年股东回报率如何？ | 无回购/分红；股东回报率持续<2% | 仓位上限×85% |
| **结构性风险（中概专用）** | 是否有VIE架构？是否在PCAOB观察名单？ | VIE架构存在；有审计摩擦记录 | 必须做"强制退市"情景压力测试；折现率+1% |
| **政策优先度** | 该公司是否曾被要求做损害股东利益的事？管理层是否有平衡政策要求与商业利益的能力/意愿？ | 无抵制记录；所在行业被政策定义为"公共服务" | 折现率+0.5%；护城河评分降一档 |

> **综合判断**：命中≥3条风险信号 → 无论巴菲特质量评分多高，总仓位上限不超过正常的50%。所有命中的风险信号必须在投资建议报告中明确记录。

**第3.5步：财报体检——"数据支不支撑？"**

> 第3步你判断"这是好资产"，第3.5步用财报数据**验证**这个判断。个人投资者不需要做完整建模，但需要看懂**五张核心体检卡**。
>
> 完成本步取数后，回到第2.5.2执行「取数后红线」（生存/财报可信度）。命中 → 直接否决，不进入第4步估值。

**体检卡一：增长能力——"公司还在长大吗？"**

| 指标 | 看什么 | 健康标准 | 危险信号 |
|------|--------|---------|---------|
| 营收增速（YoY） | 最近4个季度的同比增速趋势 | 稳定增长型>10%；快速增长型>20% | 连续2个季度增速下滑 |
| 归母净利润增速 | 核心盈利增长，直接决定内在价值增减 | 正增长且趋势稳定 | 连续下滑或由盈转亏 |
| 核心业务营收占比 | 主业收入占总收入的比例 | >70%且稳定 | 主业萎缩、靠非经常性收入撑 |

> **关键认知**：净利润增速 > PE扩张速度 → 越涨越便宜（价值增长快于价格）；净利润增速 < PE扩张速度 → 越涨越贵（泡沫在膨胀）。注意要和估值（PE）变化对比看，而非直接和股价涨幅比。

**体检卡二：赚钱能力——"收入能变成利润吗？"**

| 指标 | 看什么 | 健康标准 | 危险信号 |
|------|--------|---------|---------|
| 毛利率 | 产品/服务的基础盈利能力 | 稳定或上升 | 连续下降→定价权丧失或成本失控 |
| 经营利润率（OPM） | 扣除运营费用（研发+销售+管理="人吃马喂"）后的利润率，反映组织效率 | >10%（平台型可更高） | 转负或大幅下滑 |
| 净利润/经调整净利润 | 最终赚了多少 | 正且增长 | 由盈转亏、亏损扩大 |
| ROE（净资产收益率） | 净资产盈利效率，巴菲特最看重的指标 | 长期>15%且稳定 | 持续下降或靠高杠杆制造假高ROE |

> **三种利润指标怎么选？**
> | 指标 | 含义 | 适用场景 |
> |------|------|---------|
> | 经营利润（OPM） | 营收减去所有运营费用，最"干净" | 判断核心业务赚不赚钱 |
> | 归母净利润 | 含非经常性损益（卖资产、投资浮盈浮亏） | 传统公司PE估值基准 |
> | 经调整净利润 | 加回股权激励、摊销、投资浮亏等非现金项 | 互联网/科技公司PE估值基准（如美团、腾讯） |
>
> **注意**：经调整净利润是**公司自己定义的**，标准不统一，有些公司会"美化"。三个都看，但判断业务健康用OPM最可靠。

> **ROE进阶**：ROE = 净利率 × 周转率 × 杠杆倍数（杜邦分析）。如果ROE高主要靠杠杆倍数，不算真本事。轻资产公司（美团、腾讯）看ROE更直观；重资产/高负债公司（万科、航空）看**ROIC（投入资本回报率）**更准确——ROIC > 资本成本（通常8-10%）= 公司在创造价值，ROIC < 资本成本 = 公司在毁灭价值。ROIC可在富途、雪球、东方财富等APP的"财务指标"中直接查看。

**体检卡三：现金能力——"赚的是真钱还是纸面利润？"**

| 指标 | 看什么 | 健康标准 | 危险信号 |
|------|--------|---------|---------|
| 经营现金流 | 日常经营收到的真金白银 | 持续为正，且>净利润 | 利润高但现金流差→可能是"纸面利润" |
| 自由现金流（FCF） | 经营现金流-资本开支 | 持续为正 | 持续为负→需要不断融资才能活 |
| 账上现金储备 | 能撑多久 | 足够覆盖12个月以上的运营 | 现金快速消耗+债务高企→生存风险 |

**体检卡四：负债健康——"公司扛不扛得住压力？"**

| 指标 | 看什么 | 健康标准 | 危险信号 |
|------|--------|---------|---------|
| 资产负债率 | 总负债/总资产 | <60%（轻资产平台型可更高） | >70%且持续上升 |
| 有息负债/EBITDA | 需要多少年利润才能还清有息债务 | <3倍 | >5倍→杠杆过高 |

**体检卡五：竞争格局变化——"钱花在哪了？"**

| 指标 | 看什么 | 健康标准 | 危险信号 |
|------|--------|---------|---------|
| 销售费用率变化 | 花在获客/补贴上的钱是否在增加 | 稳定或下降 | 突然飙升→说明竞争加剧或用户留不住 |
| 研发费用率 | 花在技术上的钱 | 维持合理水平（5-15%） | 骤降→未来可能失去竞争力 |
| 用户/商户关键运营数据 | 月活/交易用户/商户数/单量 | 稳定增长 | 核心运营指标下滑比利润下滑更危险 |

**第4步：安全边际——"这个价格合理吗？"**

> 第3.5步确认了公司基本面健康，第4步以**DCF为核心**判断当前定价是否合理，PE/PEG/PB等指标辅助验证。目的是**排除荒谬定价**，而非机械寻找历史折扣。核心认知：**好公司≠好投资，好公司+合理价格=好投资**。

**巴菲特式安全边际**：安全边际不只来自价格折扣，更来自业务质量本身。护城河越强、业务可预测性越高，允许的安全边际越小——不必要求所有标的打七折。

| 公司质量 | 安全边际要求（相对DCF基准情景） | 说明 |
|---------|--------------------------|------|
| 护城河极强、业务高度可预测（巴菲特顶级标准） | 当前价格低于DCF基准10-15%即可 | 质量本身提供缓冲 |
| 稳定增长但有竞争压力 | 低于DCF基准20-25% | 正常安全边际 |
| 困境反转 / 不确定性高 | 低于DCF悲观情景20%+ | 需要更大缓冲 |

**4.1 按标的类型选估值工具（速查）**

> 不同类型的公司/指数用不同的估值工具组合。**用错指标比不看指标更危险**。完整说明见「估值工具手册.md」。

| 林奇分类 | 主估值指标 | 辅助指标 | 绝对不要用 |
|---------|-----------|---------|-----------|
| 缓慢增长 | 股息率 | PE-TTM、PB | — |
| 稳定增长 | **DCF（核心）** + PE历史分位 | PEG、FCF收益率 | — |
| 快速增长 | PEG | PS、PE前瞻 | 静态PE |
| 周期型 | PB + 行业供需信号 | 产能利用率、库存周期 | **PE** |
| 困境反转 | EV/EBITDA + 现金消耗率 | PS、恢复后远期PE | 当前PE |
| 隐蔽资产 | NAV重估 | PB | PE |

**4.2 核心估值工具箱**

| 方法 | 怎么用 | 判断标准 | 适用类型 |
|------|--------|---------|---------|
| 历史PE分位 | 当前PE在过去5-10年的百分位 | <30%分位→可能有安全边际 | 稳定增长型、快速增长型 |
| PEG | PE ÷ 预期净利润增长率 | <1→增速配得上估值；>1.5→偏贵 | 稳定增长型、快速增长型 |
| PB（市净率） | 股价 ÷ 每股净资产 | PB<1→市场认为净资产会缩水 | **仅适用于：银行、保险、地产、重资产制造业、周期股**。互联网/科技公司PB天然很高，不代表贵 |
| PS（市销率） | 市值 ÷ 营业收入 | 与同行业对比，百分位<30%→相对便宜 | 亏损/早期高增长公司、互联网/SaaS |
| 股息率 | 每股分红 ÷ 股价 | >5%→有吸引力（个股）；>4%→有吸引力（红利指数） | 缓慢增长型、红利策略 |
| 自由现金流收益率 | 自由现金流 ÷ 市值 | >5%→类似年化5%的"债券"还有增长 | 盈利稳定的成熟公司 |
| EV/EBITDA | 企业价值 ÷ EBITDA | 与同行对比，通常<10倍偏便宜 | 困境反转型、重资产行业 |
| DCF（现金流折现） | **核心估值工具**：自建三情景两阶段模型（见4.4） | 低于悲观情景→充足；悲观~基准之间→视质量；接近基准→仅极强标的；高于乐观情景→不买 | 所有类型（误差大，只给区间不给精确值） |
| 压力测试 | 盈利下降30%后估值还贵不贵？ | 盈利降30%后PE仍<25倍→安全边际充足 | 所有类型 |

> **数据来源**：理杏仁（A股/港股指数百分位）、富途（个股全指标）、Simply Wall St（DCF可视化）、multpl.com（美股席勒PE）。完整数据来源表见「估值工具手册.md」第五章。

**4.3 PE使用的三大陷阱**

| 陷阱 | 表现 | 后果 |
|------|------|------|
| **周期股陷阱** | 周期型公司（航运、地产、资源）利润高点时PE最低，看似便宜 | 实际是行业顶部，买在最"便宜"的PE反而亏最多。**周期股不看PE**（见林奇分类法） |
| **亏损股陷阱** | 亏损公司PE为负，无法使用 | 困境反转型需用"恢复后远期PE"或市销率（PS）替代 |
| **一次性收益陷阱** | 靠卖资产、投资浮盈等一次性收入让利润暴增→PE骤降 | 要用**扣非归母净利润**算PE，过滤掉非经常性损益 |

> **口诀**：PE是"后视镜"——告诉你过去的估值，不告诉你未来。PE要和增速（PEG）、现金流、护城河**组合使用**，单独看PE做决策是危险的。

**4.4 DCF使用指南（自建三情景两阶段模型）**

DCF（现金流折现）的核心思想：**公司的价值 = 未来所有现金流折算到今天的总和**。本体系以**自建DCF**为主，Simply Wall St / GuruFocus / Alpha Spread等工具仅作对比参考——工具假设不透明，对比目的是发现差异，不能替代自建模型。

> **最大局限**："垃圾进，垃圾出"——增长率假设变1%，结果变20%+。DCF给的是**价值区间**，不是精确价格。第3步定性置信度越高，DCF结论可信度越高。

├── 当前标准化FCF（剔除一次性项目，[用户提供] / [工具获取]）
├── 第一阶段增长率（年1-5）：乐观 / 基准 / 悲观
├── 第二阶段增长率（年6-10）：逐步衰减至接近终值
├── 终值增长率（永续，通常2-3%，不超过GDP增速）
├── 折现率：按上方框架选定
└── 净债务（净现金为负值，[用户提供] / [工具获取]）

计算层
├── 第一阶段现值 = Σ FCFt / (1+r)^t
├── 第二阶段现值 = Σ FCFt / (1+r)^t
├── 终值现值 = FCF10×(1+g)/(r-g) / (1+r)^10
└── 企业内在价值 = 三段之和 - 净债务

输出层
├── 乐观情景内在价值
├── 基准情景内在价值（主参考）
├── 悲观情景内在价值
└── 当前股价 vs 三情景区间
     ├── 低于悲观情景 → 安全边际充足，仓位可满配
     ├── 悲观～基准之间 → 有安全边际，视质量决定仓位
     ├── 接近基准情景 → 合理估值，仅护城河极强标的考虑买入
     └── 高于乐观情景 → 定价荒谬，无论质量多好都不买
```

## 六、巴菲特卖出决策框架

> 买入有完整的五步流程，卖出同样需要框架。**"涨了多少"永远不是卖出理由。** 触发卖出的是基本面变化，而非价格变化。出现疑似卖出信号时，先回到第3步重新做质量筛选，而非直接决定卖出。

### 卖出触发条件（任意一条命中→评估是否卖出）

| 触发条件 | 说明 | 操作 |
|---------|------|------|
| **护城河被永久破坏** | 竞争优势不可逆地消失，新竞争者已实质性侵蚀市场份额，而非暂时受压 | 清仓 |
| **管理层失去信任** | 发现重大欺诈、严重资本乱配置、大规模利益输送 | 清仓 |
| **初始论点被证伪** | 买入时的核心假设（如"护城河稳固"/"增速可持续"）被事实否定 | 清仓或大幅减仓 |
| **出现更好的机会** | 组合中有更高确定性的标的，新标的的安全边际明显高于持仓 | 减仓换仓 |
| **估值进入极度泡沫** | 股价已高于DCF乐观情景30%以上，任何合理增长假设都无法支撑当前定价 | 部分减仓（不一定清仓） |

### 不应该卖出的情况（常见错误）

| 错误触发 | 为什么不该卖 |
|---------|-----------|
| 股价大幅上涨（涨了50%、100%） | 价格涨了不等于公司变差，可能是市场在纠正之前的低估 |
| 短期业绩不及预期（单季度） | 季度噪音≠护城河破坏，应回到第3步重新评估而非直接卖 |
| 市场整体下跌 | 系统性风险不是个股卖出信号；基本面未变时，下跌是加仓机会 |
| 宏观经济不确定性增加 | 除非直接且不可逆地损害该公司护城河，否则不构成卖出理由 |
| "我已经赚了足够多" | 个人心理账户与公司内在价值无关 |

### ETF简化流程（3步 + 产品结构红线）

> 宽基/行业ETF没有单一的商业模式、护城河和财报，因此**跳过第3步和第3.5步**；但仍需执行第2.5步的「ETF产品结构红线」（避免杠杆/合成/流动性陷阱）。

```
ETF简化流程：
  第1步：周期定位（加尔布雷思）→ 判断指数/行业处于什么情绪阶段
  第2步：认知校准（芒格）       → 审视自己是否有追热点/恐慌抛售的偏见
  第2.5步：产品结构红线（硬门槛）→ 确认“买得到也卖得掉”，且结构不引入额外非线性风险
  第4步：安全边际（格雷厄姆）   → 按指数类型选对估值指标，查百分位判断贵贱
```

**ETF产品结构红线（最小检查，命中任一条 → 不作为底仓/直接否决）**：

- **杠杆/反向/复杂策略**：若为杠杆/反向/期权增强/波动率等结构 → 不作为底仓（仅在对冲层、且你完全理解时才考虑）
- **合成复制/对手方风险**：复制方式涉及Swap/复杂衍生品/显著展期风险，且你无法解释清楚最坏情景 → 直接否决（不懂不做）
- **流动性与规模**：输入 `{基金规模/日均成交额/买卖价差}` 并标注 `[来源,{YYYY-MM-DD}]`；不满足你的最低阈值 → 否决（避免“卖不掉”）
- **费率与跟踪偏离**：输入 `{费率/跟踪误差或跟踪偏离}` 并标注 `[来源,{YYYY-MM-DD}]`；超出可接受区间 → 否决

**ETF定投金额与情绪阶段对照表**：

| 市场情绪阶段 | 定投策略 |
|-------------|---------|
| 恐慌/绝望 | 加大定投金额（平时2倍） |
| 冷静/温和乐观 | 维持正常定投 |
| 乐观/偏狂热 | 减少定投金额（平时一半），但不停止 |
| 极度狂热 | 暂停定投，考虑适当减仓 |

> **定投本身就是格雷厄姆"安全边际"思想在指数投资中的最佳实践**——市场高位时自然买少，低位时自然买多。
